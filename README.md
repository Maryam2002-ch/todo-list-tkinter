# 📝 To-Do List App

A simple to-do list desktop application built with Python and tkinter.  
Your tasks are automatically saved in a JSON file and loaded when you open the app.

## ✨ Features

- ➕ Add new tasks
- 🗑️ Delete tasks
- 💾 Auto-save to JSON file
- 📂 Load saved tasks on startup
- 🖥️ Simple and clean GUI with tkinter

## 🚀 How to Run

1. Make sure you have Python installed (version 3.6+)
2. Clone or download this repository
3. Run the program:

```bash
python todo.py
```
## 📝 How to Use
Type your task in the text field

Click "Add task" to save it

Select a task and click "Delete" to remove it

Your tasks are saved automatically

## 🛠️ Technologies Used
Python 3

tkinter (GUI)

JSON (data storage)
